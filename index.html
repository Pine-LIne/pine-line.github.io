INSTRUCTIONS:

Creating and generating a calendar (*.ics) event file using our web app is easy to do. Simply follow the written instructions here!

  • Step 1: Fill out all the fields and select all the drop down menus and radio buttons below.
  • Step 2: To generate your geolocation coordinates:
    • Make sure all the fields above it are filled incorrectly.
    • Click the "Generate Geolocation Coordinates" button.
    • Wait a few seconds for the coordinates to generate; they will display right below the button.
  • Step 3: Click the "Generate Calendar Event" button.
    • A generated *.ics file will download to your downloads folder.
    • The file will be named after your event name. For example, if your event is named Party, the *.ics file will be named Party.ics
  • Step 4: Open the newly generated *.ics with the calendar app of your choice (Google Calenar, Microsoft Outlook, iCal)
